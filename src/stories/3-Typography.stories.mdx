### Access methods

```js
typeScale = {
  h1: "3.05rem",
  h2: "2.4rem",
  h3: "1.9rem",
  h4: "1.25rem",
  h5: "1.1rem",
  p: "1rem",
  helperText: "0.8rem",
  copyrightText: "0.7rem"
};
```
**Example**: define an element's font-size as h1:
```js
import { typeScale } from '../utils';

export const TextElement = styled.button`
  font-size: `${ typeScale.h1 }`;
`
```
